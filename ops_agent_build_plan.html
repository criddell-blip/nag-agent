§ 02

Tech stack

Same family as FiberLog
Frontend
React 18 + Vite, deployed to GitHub Pages (same pattern as FiberLog). Shadcn-style components.
Backend
Supabase — Postgres for state, Edge Functions for ingest + rule engine, Realtime for dashboard, Scheduled Functions for cron.
Nag layer
Slack Bolt for Node.js, hosted as a Supabase Edge Function. DMs for personal nags, channel for team-visible items.
Sources
Gmail API · ClickUp API v2 · Google Calendar API. OAuth refresh tokens stored in Supabase Vault.
Classifier
Hybrid — keyword/regex rules for cheap signals, Claude API call for ambiguous emails (e.g. "is this actually urgent or boilerplate?").
Auth
Supabase Auth, single-user to start. Multi-user later if team members want their own nag streams.
